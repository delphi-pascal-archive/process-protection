Драйвер для перехвата завершения процесса с примером.

Управление: Запустите control.exe и драйвер автоматически начнет работу. 

В окне будет написано:
1) "Драйвер успешно инициализирован! Теперь можете попробовать завершить процесс с PID: (Тут PID запущенного процесса)", 
а если драйвер не начал работу, будет написано: "Ошибка инициализации драйвера!".

2) Если при отключении защиты будет написано "Защита успешно отключена!", то это значит что теперь процесс можно завершить,
если написано "Ошибка отключения защиты!", вы не сможете правильно выйти из программы (Таких случаев не случалось).

***************************************************************************************
Внимание: Драйвер тестировался только на Windows XP (SP2 и SP3).
          На других версиях Windows возможен BSOD.
          Драйвер распространяется без исходного кода.
          Драйвер написан на Windows Driver Developer Kit.
